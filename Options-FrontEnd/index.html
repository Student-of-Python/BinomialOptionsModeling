Let’s Talk About Options

A Gleb Sokhin Project

Options have always been a topic that has fascinated me. However, they’re far from simple, and they can be hard to grasp at first. Questions like “Why are options priced the way they are?” or “What is this option’s price actually telling me?” kept coming up when I first started learning.

I struggled to find resources that explained options clearly from the ground up, so I decided to create one myself. I wish I had a resource like this when I first began learning options, so if you’re curious, this website is for you.

Come along with me. Let’s talk about options.
