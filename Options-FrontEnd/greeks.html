The Greeks

How sensitive an option's value is to underlying factors.

Delta

Definition

Delta measures the sensitivity of an option’s value to changes in the price of the underlying asset. Delta shows how much the option's value will change given the underlying asset changes by $1.

Mathematical Representation

\[ \Delta = \frac{\partial V}{\partial S} \]

In the binomial model, Delta is approximated by comparing option values of its left and right children. At expiration, I defined Delta to be 1.00 when in-the-money (ITM), 0.00 when out-of-the-money (OTM), and 0.5 if at-the-money (ATM)

\[ \Delta \approx \frac{V_{\text{up}} - V_{\text{down}}} {S_{\text{up}} - S_{\text{down}}} \]

Alternate Uses & Interpretation

Hedge Ratio

Delta represents the number of shares of the underlying required to hedge one option position. A Delta of 0.6 implies holding 0.6 shares per option.

Directional Exposure

Delta captures an option’s directional bias. Calls have positive Delta, puts have negative Delta, and the magnitude reflects the option’s moneyness (its strike price relative to the underlying price)).

Probability Proxy

Under risk-neutral pricing, Delta can be interpreted as an approximation of the probability that the option expires in-the-money.

Gamma

Definition

Gamma measures the sensitivity of Delta to changes in the underlying price. If Delta is velocity, Gamma is acceleration; it determines how fast Delta changes.

Mathematical Representation

\[ \Gamma = \frac{\partial^2 V}{\partial S^2} \]

In a binomial framework, Gamma is approximated using changes in Delta across price nodes. I did not define Gamma at terminal nodes.

\[ \Gamma \approx \frac{\Delta_{\text{up}} - \Delta_{\text{down}}} {S_{\text{up}} - S_{\text{down}}} \]

Alternate Uses & Interpretation

Curvature

Gamma captures how non-linear an option is. A stock's value changes linearly with its price, so its Gamma is zero. An option's value changes non-linearly as the underlying price moves, and Gamma measures the strength of that bend.

Convexity Risk

A positive Gamma means Delta will move proportionally with the underlying price, while a negative Gamma means Delta will move inversely proportional. This means long options benefit from large price moves due to positive Gamma, while short options have negative Gamma and are harmed by volatility.

Re-hedging Frequency

High Gamma means Delta changes rapidly, requiring frequent rebalancing to remain Delta-neutral. The portfolio needs to change more often with a higher Gamma to re-hedge.

Theta

Definition

Theta measures how an option’s value changes as time passes, holding all else constant. It quantifies time decay and and measures uncertainty.

Mathematical Representation

\[ \Theta = \frac{\partial V}{\partial t} \]

In the binomial model, Theta is approximated using midpoint traversal (going left, right after a parent node). Theta is not defined at nodes which lack a middle node.

\[ \Theta \approx \frac{V_{\text{mid}} - V_{\text{now}}}{2\,\Delta t} \]

Alternate Uses & Interpretation

Time Decay

As expiration approaches, uncertainty collapses and option value erodes. Theta accelerates near maturity since the underlying's price is more certain with less time. Theta decay rapidly eats away an option's value close to expiration.

Cost of Waiting

Traders can lose despite being directionally correct; Traders must enter a position enough to overcome Theta.

Theta–Gamma Tradeoff

The more an option can gain from sudden moves, the most it costs to hold over time. If the market does not move, time works against you.

Vega

Definition

Vega measures sensitivity to volatility. It reflects how option prices respond to changes in uncertainty, not direction.

Mathematical Representation

\[ \text{Vega} = \frac{\partial V}{\partial \sigma} \]

In the binomial model, Vega is computed using small changes in volatility. Vega is a property of the tree, therefore only the root node can have Vega.

\[ \text{Vega} \approx \frac{V(\sigma + \varepsilon) - V(\sigma - \varepsilon)} {2\,\varepsilon} \]

Alternate Uses & Interpretation

Volatility Exposure

Options with positive Vega gain when volatility rises, regardless of movement's direction. What matters is the size of the movement, not the direction.

Event Risk

Earnings and macro events increase implied volatility, creating opportunities. However, after the event, volatility often declines, taking the option's value down with it (IV Crush).

Time Dependence

The more time an option has till expiration, the more sensitive it will be to changes in implied volatility. With more time, uncertainty has a greater impact on the option's value.

Rho

Definition

Rho measures sensitivity to the risk-free interest rate. It quantifies how discounting future payoffs impacts option value.

Mathematical Representation

\[ \rho = \frac{\partial V}{\partial r} \]

In the binomial model, Rho is computed using small changes in risk free rate. Rho is a property of the tree, therefore only the root node can have Rho.

\[ \rho \approx \frac{V(r+\varepsilon) - V(r-\varepsilon)} {2\,\varepsilon} \]

Alternate Uses & Interpretation

Time Value of Money

Money received today is worth more than money received in the future. When interest rates increase, future option payoffs are discounted more heavily, reducing their present value.

Long-Dated Sensitivity

Changes in interest rates impact the options value less when coming closer to expiration since there is less time for discounting and therefore less emphasis on the time value of money.

Macro Hedging

When interest rates change, option prices change. Traders hedge by using other instruments such as bonds and rate futures.
